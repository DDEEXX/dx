Файлы начинающиеся с точки игнорируются

Структура скрипта:
файл должен содержать класс с именем совпадающим с именем файла, но без расширения .php
Класс должен содержать статический метод start()

Пример.
Имя файла:
test_pause5.php
Содержимое файла:
<?php
class test_pause5
{
    static function start() {
        sleep(5);
    }
}
